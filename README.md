# YOLO Vehicle Detection

This project detects vehicles (cars, trucks, buses, motorbikes) in **images** and **videos** using YOLOv10x and YOLOv8n.

## Requirements
- Python 3.8+
- PyTorch
- OpenCV
- Matplotlib
- Pandas
- Ultralytics YOLO

Install dependencies:
```bash
pip install -r requirements.txt
